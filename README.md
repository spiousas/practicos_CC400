## Course Website

This repository contains the source files for the CC400 - Experimental and Quasiexperimental Design course website, built with Quarto.

The website is automatically published to GitHub Pages when changes are pushed to the main branch.

### Local Development

To build the website locally:

1. Install Quarto from https://quarto.org/docs/get-started/
2. Clone this repository
3. Run `quarto render` to build the site
4. The generated website will be in the `_site` directory

### Adding New Assignments

To add a new assignment:

1. Create a new `.qmd` file in the `assignments/` directory
2. Update `assignments/index.qmd` to include a link to the new assignment
3. Commit and push your changes

The website will be automatically rebuilt and deployed.
